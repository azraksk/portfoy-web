Kişisel Web Sitesi Projem

Bu proje, HTML, CSS ve JavaScript kullanarak geliştirdiğim kişisel portföy web sitesidir. Hakkımda, projelerim ve iletişim bölümleri içerir. Kullanıcılar form aracılığıyla bana doğrudan mesaj gönderebilir.

Kullanılan Teknolojiler

  • HTML5
  • CSS3 (Responsive tasarım)
  • JavaScript (Popup mesajlar ve form gönderimi)

Linkler

  • GitHub Proje Sayfası
  • Canlı Demo

Öğrendiklerim

Bu projede kullanıcı arayüzü tasarımı, responsive düzen kurma ve form verilerini düzgün şekilde işleme konusunda deneyim kazandım. Ayrıca JavaScript ile basit kullanıcı geri bildirimleri (popup gibi) sunmayı öğrendim.
